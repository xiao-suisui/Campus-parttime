## 6. 执行顺序（建议）

1. 先迁移 `service` 与 `mapper` 包路径（IDE 重构 + 编译）。
2. 再迁移 `controller` 包路径并调整 `@RequestMapping`。
3. 最后迁移 `domain` 与 `mapper xml` 路径。
4. 不保留旧兼容层，迁移完成后直接删除旧路由与旧目录。
