# 联调测试与收尾计划

> 生成日期：2026-05-06
> 前置依赖：PC 端全部功能已完成（阶段一~三），移动端已完成（阶段四）

## 本阶段目标

完成全链路联调测试，修复发现的问题，确保系统可正常运行。

---

## 任务 5.1：后端编译验证

**目标**：确保所有后端代码可正常编译

**操作**：
```bash
cd backend && mvn -DskipTests clean compile
```

检查是否有编译错误，修复 import 缺失、类型不匹配等问题。

---

## 任务 5.2：前端编译验证

**目标**：确保前端代码可正常编译

**操作**：
```bash
cd frontend && npm run dev
```

检查是否有编译警告或错误，修复组件注册、API 导入等问题。

---

## 任务 5.3：RBAC 脚本执行

**目标**：执行统一 RBAC 初始化脚本

**操作**：
1. 确认数据库 `campus_parttime` 可连接
2. 执行 `backend/sql/init/init_all_rbac.sql`
3. 验证菜单、权限、角色绑定数据正确

---

## 任务 5.4：接口联调测试

**目标**：验证各角色 API 可正常调用

**测试清单**：

### 学生端
- [ ] 登录：`POST /login`（用户名/密码）
- [ ] 岗位列表：`GET /student/post/list`
- [ ] 岗位详情：`GET /student/post/{postId}`
- [ ] 岗位推荐：`GET /student/post/recommend`
- [ ] 投递：`POST /student/application`
- [ ] 投递列表：`GET /student/application/list`
- [ ] 收藏：`POST /student/collection`
- [ ] 收藏列表：`GET /student/collection/list`
- [ ] 取消收藏：`DELETE /student/collection/{postId}`
- [ ] 评价列表：`GET /student/evaluation/list`
- [ ] 提交评价：`POST /student/evaluation`
- [ ] 简历查询：`GET /student/resume`
- [ ] 简历保存：`POST/PUT /student/resume`
- [ ] 个人信息查询：`GET /student/info`
- [ ] 个人信息保存：`POST/PUT /student/info`
- [ ] 公告列表：`GET /system/notice/listTop`

### 企业端
- [ ] 企业信息：`GET/POST/PUT /enterprise/info`
- [ ] 岗位管理：`GET/POST/PUT/DELETE /enterprise/post`
- [ ] 投递管理：`GET /enterprise/application/list`
- [ ] 录用/拒绝：`PUT /enterprise/application/{id}/accept` `/reject`
- [ ] 完成标记：`PUT /enterprise/application/{id}/complete`

### 管理端
- [ ] 企业审核：`GET /admin/enterprise-audit/list`
- [ ] 审核通过/驳回：`PUT /admin/enterprise-audit/{id}/approve` `/reject`
- [ ] 岗位监管：`GET /admin/job-review/list`
- [ ] 强制下架：`PUT /admin/job-review/{id}/force-down`
- [ ] 运营统计：`GET /admin/stat/overview` `/trend`

---

## 任务 5.5：前端页面验证

**目标**：在浏览器中验证各页面可正常渲染和交互

**操作**：
1. 启动后端：`cd backend && java -jar ruoyi-admin/target/ruoyi-admin.jar`
2. 启动前端：`cd frontend && npm run dev`
3. 分别以 student/enterprise/admin 角色登录测试

---

## 任务 5.6：问题修复

**目标**：修复联调中发现的问题

**预期问题**：
- 权限不足（403）：检查 RBAC 脚本是否执行
- 接口返回空：检查数据库数据
- 前端报错：检查 API 路径、组件注册

---
