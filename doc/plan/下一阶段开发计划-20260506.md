## 任务 3.1：智能推荐模块（创新点 1）

**目标**：为学生提供基于规则的岗位推荐

### 3.1.1 数据库：岗位标签体系

**简化方案**（在现有表中加字段，避免新建多张表）：
- `t_job_post` 新增 `skill_tags varchar(500)` 字段（岗位技能标签）
- `t_student_resume` 已有 `skill_tags` 字段（学生技能标签）
- `t_student_info` 新增 `expected_position varchar(100)` 字段（求职意向）

**SQL**：编写 ALTER TABLE 脚本

### 3.1.2 后端：推荐服务

**新建**：
- `backend/ruoyi-system/.../biz/student/service/IRecommendService.java`
- `backend/ruoyi-system/.../biz/student/service/impl/RecommendServiceImpl.java`

**核心方法**：
```java
List<JobPost> recommendPosts(Long studentId, int limit)
```

**匹配逻辑**（基于规则，非机器学习）：
1. 读取学生简历的 `skill_tags` 和 `expected_position`
2. 读取学生历史投递/收藏的岗位类型
3. 查询上架岗位，按以下维度打分：
   - 技能标签匹配度（Jaccard 相似度）
   - 求职意向与岗位名称/标签的关键词匹配
   - 薪资范围匹配
   - 工作类型偏好
4. 按总分降序排列，返回 top N

**端点**：
- 在 `StudentJobController` 中新增 `GET /student/post/recommend` 端点
- 权限：`student:job:recommend`

### 3.1.3 前端

- 新建 `frontend/src/api/student/recommend.js`
- 在 `views/system/student/post/index.vue` 增加"推荐"标签页或顶部推荐区域

### 3.1.4 SQL

- 新增 `student:job:recommend` 权限
- 字段变更 ALTER TABLE 脚本
